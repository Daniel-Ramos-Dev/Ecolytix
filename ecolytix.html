Sobre o Ecolytix
Plataforma IoT ambiental — Fundação 404
🌿 Missão
O Ecolytix é uma plataforma open-source de monitoramento ambiental desenvolvida pela Fundação 404. O sistema conecta sensores Arduino físicos via USB diretamente ao navegador usando a Web Serial API, exibindo dados ambientais reais em tempo real — sem intermediários, sem dados inventados.
Princípio fundamental: "Se não houver dado real do sensor ou inserido por usuários verificados, o sistema exibe 'sem dados disponíveis' — nunca inventa valores."
📡
Sensor Arduino MQ
Módulo MQ lê qualidade do ar via pino analógico A0. LEDs indicam status localmente. Dados enviados via Serial a 9600 baud no formato CSV.
🔌
Web Serial API
API nativa do Chrome permite conexão USB direta entre o Arduino e o browser sem instalar drivers adicionais. Suporte em Chrome 89+.
💾
Fallback localStorage
Quando o sensor está offline, o sistema recupera o último dado real armazenado e informa o usuário claramente, sem exibir dados falsos.
👥
Dados Comunitários
Usuários autenticados podem registrar observações ambientais manuais. Esses dados são tratados como contribuições humanas verificadas.

Serial.begin(9600); // Saída CSV: AQI,STATUS Serial.print(avg); // valor bruto médio Serial.print(","); Serial.println(status); // VERDE / AMARELO / VERMELHO // Limites internos do sensor: // VERDE ≤ 300 // AMARELO ≤ 450 // VERMELHO > 450
